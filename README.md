# ruby-box

[Azk](http://azk.io) box for ruby.

## Versions

* Ruby 2.0.0

## Example

Using a box is trivial, the azkfile.json includes a box definition that you can leverage.

```json
"box": "azukiapp/ruby-box#0.0.1",
```
